# Bioinformatics Learning Series Volume 1

## Amplicon Analysis and QIIME2-to-R Workflows

This repository is part of the Bioinformatics Learning Series developed by Jojy John.

Volume 1 focuses on amplicon-based microbiome analysis, including QIIME2 workflows, visualization, functional prediction, and QIIME2-to-R integration.

The goal is to provide practical tutorials, scripts, and workflows that can be adapted for microbiome studies using 16S, 18S, ITS, COI, and 12S sequencing data.

---

## Repository Structure

### 01_amplicon_week

A complete introductory amplicon analysis series covering:

- Introduction to amplicon sequencing
- QIIME2 workflows
- Classifier training
- Contamination removal
- Rarefaction strategies
- Visualization
- Functional prediction
- Web-based microbiome analysis

### 02_qiime2_to_r

Resources for exporting QIIME2 outputs into R for downstream analysis and visualization.

Contents include:

- QIIME2 visualization guide
- Alpha and beta diversity statistics
- Patchwork-based figure generation
